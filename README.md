# 🎬 Netflix SQL Data Analysis

## 📌 Overview

This project analyzes the Netflix titles dataset using MySQL. It focuses on exploring the data and solving business-oriented questions through SQL queries involving filtering, aggregation, window functions, and string manipulation.

## 🛠️ Tools & Technologies

* **MySQL**
* **SQL**

## 📂 Project Workflow

1. Created the `netflix_db` database and `netflix` table.
2. Imported the Netflix dataset using `LOAD DATA INFILE`.
3. Performed initial data exploration to understand the dataset.
4. Solved multiple business problems using SQL queries and analytical functions.

## 🔍 Key Analyses

* Counted the number of Movies vs TV Shows.
* Identified the most common content ratings.
* Retrieved movies released in a specific year.
* Found the top countries with the most Netflix content.
* Identified the longest movie in the dataset.
* Retrieved content added within the last five years.
* Listed content directed by a specific director.
* Found TV Shows with more than five seasons.
* Counted content across different genres.
* Analyzed Indian content added over the years.
* Counted documentary movies.
* Identified records with missing director information.
* Analyzed appearances of a specific actor.
* Found top actors appearing in Indian movies.
* Categorized content based on keywords in descriptions.

## 💡 SQL Concepts Used

* Common Table Expressions (CTEs)
* Window Functions (`RANK`, `LAG`, `CUME_DIST`)
* `JSON_TABLE`
* Date Functions
* String Functions
* Regular Expressions (`REGEXP`)
* Aggregations and Grouping
* Conditional Logic with `CASE`
